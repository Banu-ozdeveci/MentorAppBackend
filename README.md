# MentorAppBackend
Backend for mentorApp using Node.js, Express, MongoDb, Redux



# Technology Used 

 I have built this project using the following tools & techniques:

- Node
- Express
- MongoDB
- -Redux

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```


## Start

To run this project:

```
$ cd ../projectName
$ npm start
```
